دستورات سوپرگپ📄:
!info
نشان دادن اطلاعات گپ📄
!admins
لیست ادمین ها📄
!owner
لیست ادمین های گپ📄
!bots
نشان دادن ربات های داخل سوپر گپ📄
!who
نشان دادن لیست کاربران گروه📄
!block
بلاک کردن کسی از سوپر گروه با ریپلای📄
*Adds user to blocked list*
!ban
بن کردن کسی از گروه با ریپلای📄
!unban
آنبن کردن کسی از گروه با ریپلای
!id
نشان دادن آیدی عددی گروه📄
!kickme
اخراج کردن شاما از گپ📄
!setowner
ادمین گروه کردن کسی با ریپلای📄
!setname
تغییر دادن اسم گپ📄
!setphoto
تغییر دادن عکس گپ
!setrules
تنظیم قوانین📄
!newlink
ساخت لینک جدید برای گپ📄
!link
دریافت لینک📄
!rules
نمایش قوانین
!lock [links|flood|spam|Arabic|member|rtl|sticker|contacts|strict]
قفل کردن با انتخاب هر آیتم📄
!unlock [links|flood|spam|Arabic|member|rtl|sticker|contacts|strict]
برداشتن قفل هر آیتم📄
*rtl: Delete msg if Right To Left Char. is in name*
*strict: disable strict settings enforcement (violating user will not be kicked)*
!mute [all|audio|gifs|photo|video|service]
سکوت کردن یک آیتم📄
*A "muted" message type is auto-deleted if posted
!unmute [all|audio|gifs|photo|video|service]
برداشتن حالت سکوت هر آیتمs📄
*A "unmuted" message type is not auto-deleted if posted
!setflood [عدد]
تنظیم تعداد پیام های غیر مجاز📄
!settings
نشان دادن تنظیمات گروه📄
!muteslist
نشان دادن لیست سکوت📄
!muteuser [username]
ساکت کردن یک فرد با ریپلای📄
*If a muted user posts a message, the message is deleted automaically
*only owners can mute | mods and owners can unmute
!banlist
لیست بن شده ها📄
!clean [rules|about|modlist|mutelist]
!del
پاک کردن پیغام خاصی با ریپلای
*Search for kick reasons using [#RTL|#spam|#lockmember]
**You can use "#", "!", or "/" to begin all commands
*Only owner can add members to SuperGroup
(use invite link to invite)
*Only moderators and owner can use block, ban, unban, newlink, link, setphoto, setname, lock, unlock, setrules, setabout and settings commands
*Only owner can use res, setowner, promote, demote, and log commands
